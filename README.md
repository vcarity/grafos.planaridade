# Planaridade

Se um grafo G é um grafo planar. Então E <= 3V - 6;

# Instruções #

1. Abra o terminal
2. Vá até o diretório onde estão localizados os arquivos
3. Compile com:

'''
javac \*.java
'''

4. Rode com:

'''
java Main entrada
'''

Onde :

- entrada = arquivo texto de entrada

Lembrete: Os arrays em java começam em 0.
